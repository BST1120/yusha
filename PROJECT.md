# 『遊者』プロモーションサイト プロジェクト定義書

## 1. プロジェクト名

**遊者 / YUSHA - Physical Edition Web Archive**

## 2. プロジェクトの目的

本プロジェクトは、コンセプトアルバム『遊者』の物体版CDに封入するQRコードからアクセスできる、作品補完用Webサイトを制作するものである。

このWebサイトは、単なる歌詞掲載ページではなく、CD・ジャケット・ステッカー・封入紙と連動し、アルバムの世界観をWeb上に拡張するためのアーカイブとして設計する。

主な目的:

1. 物体版CDの購入者が、歌詞・制作年表・制作メモにアクセスできるようにする
2. 購入者限定の音源ダウンロードページへ誘導する
3. アルバム『遊者』の思想・世界観を補完する
4. QRコードからスマホで読みやすくアクセスできるページにする
5. 将来的にプロモーションサイトとしても使えるようにする

---

## 3. アルバム概要

- **アルバムタイトル**: 遊者
- **コンセプト**:
  - 正解に依存せず、自分で正解を作り出していく
  - 正解とは、遊びの中で見つかっていくもの
  - 遊んでいる時の思考を優先する
  - 遊んでいる者が世界を変える

タイトルトラック「遊者」には「遊んでいるものよ集え、世界をともに変えよう」というメッセージが含まれている。

Webサイト上ではメッセージを直接的に説教っぽく見せず、**記録・アーカイブ・封入資料・制作ログ**のように提示する。

## 8. サイトの基本思想

宣伝サイトではなく、**封入紙から続くWeb上の記録室**として扱う。

```text
CDを買った → 紙ジャケを開いた → 封入紙を見た → QRからWebアーカイブに入る → 歌詞・制作メモ・ダウンロードへ進む
```

派手な広告ページではなく、**作品世界の奥にある資料室・作業机・記録庫**のように設計する。

---

## 9. Webサイト要件（概要）

- **実装**: 静的サイト（HTML / CSS / JavaScript）
- **ホスティング**: GitHub Pages（第一候補）、Netlify / Cloudflare Pages（代替）
- **ページ**: `index.html` / `lyrics.html` / `archive.html` / `download.html` / `player.html`
- **音源方針**: GitHub Pages には試聴用プレビューMP3のみ。フル音源ZIPは外部ストレージ（Google Drive / Dropbox 等）
- **QR導線**: 封入紙QR1＝トップ、QR2＝ダウンロード（推奨）

詳細は `README.md` および `docs/web-requirements.md` を参照。
