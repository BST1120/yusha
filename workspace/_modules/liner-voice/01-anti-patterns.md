# 書き手臭い — NG パターンと直し方

## 臭いの正体

読者が「作者の論考を読まされている」と感じる状態。  
ライナーノーツは **資料** に近く、エッセイや評論ではない。

---

## NG パターン

| パターン | 例 | なぜ臭い |
|----------|-----|----------|
| **問いかけの連打** | 「〜ではないだろうか」「どうだろうか」 | 読者に答えを求める |
| **意図の宣言** | 「私は〜を意識していた」 | 説明が先に立つ |
| **抽象の積み上げ** | 設計、内側、外側、メッセージ性… | 事実が見えない |
| **見出しで論を構築** | 【当時の意図】【位置づけ】 | 教材・評論の形 |
| **メタな予告** | 「今後のテーマとして…」 | 記事の外を語る |
| **同義反復** | 同じ結論を段落で言い直す | 冗長・説教的 |

---

## OK パターン（ライナー／インタビュー調）

| パターン | 例 |
|----------|-----|
| **冷たい事実から入る** | 「Suno に千文字プロンプトを入れた。返ってきたのをそのまま Cubase に読んだ。」 |
| **手順** | 「Chopping → Groove Agent → 再配置。この3段。」 |
| **一言エピソード** | 「完成したとき、すごくうれしかった（JJJ 系）。」 |
| **自白・ノリ** | 「ほんまにノリで出来た曲（百足系）。」 |
| **具体数字・固有名詞** | FL Studio、8080、2024/02、Groove Agent |
| **短い段落** | スマホで3スクロール以内に一段落完結 |

---

## 言い換え表

| NG | OK |
|----|-----|
| 〜と考えている | （削除）または「〜だった」 |
| 〜を意識していた | 「〜した」「〜だけ使った」 |
| 〜ではないだろうか | 削除。事実だけ残す |
| 【当時の意図】 | 見出しなし。段落で事実 |
| 設計に対して内側にある | 「タイトルは後からつけた。曲は先にあった。」 |

---

## inner image 第2稿で該当しやすい箇所

- 「〜と考えている」が複数
- 「メダルに値するのではないだろうか」→ エッセイの締め
- AI セクション + 位置づけセクションで **同じ結論を二回**

**直し方:** AI は3文。後付けはエピソード1つ。残りは Suno → Cubase → チョップの順番。
